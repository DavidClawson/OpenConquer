# Screenshots

Drop PNG screenshots here to show them in the top-level `README.md`. The repo's
`.gitignore` ignores `*.png` everywhere **except** `docs/screenshots/` (and
`docs/**/*.png`), so images placed here are committable while extracted game art
stays ignored.

The main README references these filenames — add images with these names and
they'll render automatically:

| Filename | Suggested shot |
|----------|----------------|
| `gameplay.png` | An in-game view (base + units + HD sprites) |
| `sidebar.png`  | The sidebar / build UI, ideally mid-mission |
| `options.png`  | The Options screen (Classic vs Enhanced ruleset) |

Tips:
- Capture at a decent window size; the engine supports arbitrary resolutions.
- PNG keeps the crisp pixel/HD art. Keep files reasonably sized (< ~1–2 MB each).
